🎓 三段教学法（核心方法论）
  • 第一步：故事情景
    用学生能理解的生活例子讲解知识点，回答"为什么需要这个"
  • 第二步：代码实现
    给出最简单的代码示例，通过输出结果让学生看到魔法发生
  • 第三步：学生动手
    给学生改进任务，让他们自己改代码、自己写程序
🎯 五大教学策略
1️⃣ 对比学习法
展示"坏方案"和"好方案"的对比，让学生理解为什么需要某个语法
2️⃣ 可视化调试
用图表、动画展示变量���值如何变化，帮助理解程序执行流程
3️⃣ 错误预见法
提前告诉学生常见错误，让他们有意识地避免
4️⃣ 项目驱动
每个知识点都关联一个有趣的小项目，学生在做项目中学习
5️⃣ 成就感体系
定期展示进度、小竞赛排名、表扬创意代码，激励学生
📅 标准课堂节奏（每周安排）
  • 第1课（90分钟）：新知识讲解 → 故事情景 → 代码演示 → 学生第一次操作
  • 第2课（60分钟）：练习与答疑 → 学生自己写代码 → 常见错误讨论
  • 第3课（60分钟）：项目实战 → 综合应用 → 展示作品
  • 第4课（30分钟）：小竞赛或复习 → 排名展示 → 激励表扬
⭐ 入门期（第1-4周）| 目标：让学生爱上编程
  • 第1周：编程思维启蒙
    用生活例子（做三明治、去学校、玩游戏的"程序"）让学生理解程序的本质：有步骤、有顺序、有条件、有重复
  • 第2周：第一个程序
    cout << "我叫XXX，我要成为竞赛高手！" << endl;
    让学生第一次看到程序执行的结果，获得"哇！"时刻
  • 第3周：变量概念
    不讲"存储数据的容器"，而讲"快递员用标签记录信息"。学生理解为什么需要变量
  • 第4周：基础输入输出
    学生能写简单程序：输入名字和年龄，输出个人信息。完成第一个"作品"
📝 第1周详细教案

课程内容：编程思维启蒙

时长：90分钟 | 难度：⭐ | 工具：无需电脑
  • 0-10分：引入
    "你们有没有玩过游戏？游戏是怎么玩的？"
    → 引出"游戏有规则"、"每个动作都有意义"
  • 10-30分：故事讲解
    案例1：做一个三明治的"程序"
    1. 拿出面包
    2. 涂上黄油
    3. 放上生菜和火腿
    4. 盖上另一片面包
    "这4步缺一不可，顺序也很重要��这就是一个'程序'！"
  • 30-50分：学生观察
    老师给出3个例子，学生一起分析：
    • 去学校的程序
    • 玩游戏的程序
    • 做作业的程序
    "你们看，这些'程序'都有什么共同点？"
  • 50-70分：程序四要素
    ✓ 有步骤：必须一步一步来
    ✓ 有顺序：顺序很重要
    ✓ 有条件：有时候"如果...就..."
    ✓ 有重复：有时候要做同一件事多次
  • 70-90分：计算机程序
    "计算机就是一个'奴隶'，它按照我们给它写的'程序'执行。程序写得好，计算机就听话！"
    展示简单的游戏或APP，让学生看到程序的魔力
⭐⭐ 初级期（第5-12周）| 目标：建立基础思维
  • 第5-6周：条件判断（if-else）
    场景：成绩分级。学生理解"条件决定行动"
  • 第7-8周：循环（for）
    场景：抄写100遍。让学生感受循环的威力：避免重复代码
  • 第9-10周：循环进阶（while）+ 嵌套循环
    学生能写99乘法表、画图形、做简单游戏
  • 第11-12周：数组和基础算法
    学生能处理多个数据、做简单的排序和查找
📝 第5周详细教案：if-else 条件语句

核心概念：根据条件做不同的动作

时长：90分钟 | 难度：⭐⭐ | 项目：成绩分级系统
  • 故事情景：
    "你在便利店买东西。如果钱够，就买；如果钱不够，就回家。这就是'条件判断'！"
  • 代码演示：
    if (score >= 90) cout << "优秀"; else cout << "加油";
  • 学生动手：
    1. 输入一个数字，判断奇偶
    2. 输入年龄，判断是否能看电影
    3. 输入三个数字，找最大值
⭐⭐⭐ 进阶期（第13-20周）| 目标：进阶竞赛思维
  • 第13-14周：函数
    核心：代码复用、模块化。学生能自己定义函数、参数传递、返回值
  • 第15-16周：字符串处理
    学生能做文本处理：分割字符串、查找子串、字符串反转
  • 第17-18周：递归初步
    学生理解递归思想，能写简单的递归程序（阶乘、斐波那契）
  • 第19-20周：二维数组和矩阵操作
    学生能处理表格数据、矩阵转置、游戏地图
🏆 竞赛期（第21周+）| 目标：竞赛实战
  • 第21-22周：动态规划基础
    经典问题：硬币问题、背包问题
  • 第23-24周：搜索和回溯
    经典问题：DFS、BFS、八皇后问题
  • 第25周+：贪心和图论基础
    学生能解决竞赛题目，参加区级或全国竞赛
